Detailed Description

template<typename Tin = std::vector<double>>
class Sink< Tin >

Base class for sinks

This class is the base class for all sinks. It defines the interface for loading data and processing it. Derived classes must implement Sink::kind, Sink::load_data and Sink::process methods. Optionally, they can implement the Sink::set_params method to receive parameters as a void pointer.

After deriving the class, remember to call the INSTALL_SINK_DRIVER(klass, type_in, type_out) macro to enable the plugin to be loaded by the kernel.
